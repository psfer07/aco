# Búsqueda de Rutas de Evacuación Optimizadas basado en Algoritmo de Colonia de Hormigas y A* Search

En este repositorio de recogen el código fuente desarrollado para el proyecto de investigación.
Se puede acceder a la versión estable de forma online a través del siguiente enlace:
[Link al simulador](https://psfer07.github.io/aco)

---
## Versión actual v0.1
* Se ha creado una interfaz web de usuario para conectar en tiempo real la simulación de los algoritmos con una reppresentación gráfica
* Se han añadido un menú de configuraciones donde el usuario podrá modificar algunos parámetros del simulador
  * Un control deslizante para alternar entre usar o no las ventanas como una salida
  * Un control deslizante para mostrar objectos en la habitación
    * Hasta ahora los objetos mostrados son mesas y unos pilares a los lados de la habitación para simular una clase de instituto
  * Un control deslizante para mostrar o no una malla que indique la posición exacta de las celdas
